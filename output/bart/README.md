# <img src="logo.jpeg" height="20"> Bart

Retrieve real-time BART (Bay Area Rapid Transit) train departure estimates, plan trips between stations, and access schedule, fare, route, and station information. Look up service advisories, elevator outage status, and active train counts. Get fare costs between stations, station access details including parking and bike facilities, and holiday or special schedules. All data is read-only.
