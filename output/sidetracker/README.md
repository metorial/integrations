# <img src="logo.png" height="20"> Sidetracker

Track and analyze marketing performance with cookie-free website analytics. Retrieve and manage conversion lists and their rows. Attach custom metadata to visitor sessions for cross-domain tracking enrichment. Add revenue data to sessions for marketing attribution. Update sales status on customer sessions to track pipeline progress. Track call-to-action events programmatically via the API. Receive real-time webhook notifications when list rows are created or updated.
