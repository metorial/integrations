# Faceup

Monitor and manage whistleblower reports and compliance cases. Query report statistics by date range, retrieve case details and statuses, access internal investigator comments, and track anonymous messages between reporters and case handlers. Receive real-time webhook notifications when new reports are submitted, new messages arrive, or internal comments are added. Integrate case data with external project management and collaboration tools for automated task creation and alerting.
