# <img src="logo.png" height="20"> Callpage

Manage callback automation for website visitors. Initiate immediate or scheduled phone calls through embeddable widgets, retrieve and filter call history with detailed metadata (duration, recording, geo-location, UTM data, tags, notes). Create and configure widgets with call algorithms, caller ID modes, recording settings, language, and after-hours behavior. Manage users (agents) and managers with configurable business hours and department assignments. Customize SMS templates and voice greetings per widget. Receive webhooks for call events including scheduled, new, missed, completed calls, and visitor messages.
