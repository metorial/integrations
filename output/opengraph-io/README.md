# Open Graph.io

Extract Open Graph metadata, Twitter Cards, and HTML meta tags from any URL to generate link previews. Scrape raw HTML from websites with JavaScript rendering and anti-bot proxy support. Capture screenshots of web pages at various dimensions and quality levels. Extract structured content from pages for AI/LLM pipelines. Ask AI-powered questions about any webpage and receive structured answers based on page content. Supports caching, full browser rendering, proxy routing, and language preferences.
