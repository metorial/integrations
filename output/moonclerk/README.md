# <img src="logo.png" height="20"> Moonclerk

Retrieve payment, customer, and form data from MoonClerk's Stripe-based payment processing platform. Read payment transactions filtered by form, date range, customer, or status (successful, refunded, failed). Read customer and subscription (plan) data filtered by form, checkout date, next payment date, or status (active, canceled, expired, past_due, pending, unpaid). List and retrieve payment form configurations. Receive real-time webhook notifications for payment created, plan created, plan ended, and plan payment failed events. The API is read-only — no creation, modification, or deletion of records is supported.
