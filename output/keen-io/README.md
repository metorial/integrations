# Keen.io

Collect, store, and analyze custom event data via API. Record single or batch events into collections, run analytical queries (count, sum, average, percentile, funnel analysis), and extract full-resolution data. Supports saved and cached queries, cached datasets for sub-second lookups, data enrichment add-ons (IP-to-geo, user agent parsing, URL parsing, datetime parsing), schema inspection, access key management, and streaming data to Amazon S3. Filter, group, and segment analyses by timeframe, interval, and property values.
