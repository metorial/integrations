Manage self-storage, valet storage, and mobile storage operations. Create and manage storage sites, units, and tenant accounts. Handle move-in and move-out orders, valet storage orders, and unit transfers. Create and track invoices, process payments, and manage recurring billing. Monitor unit availability, occupancy, and rental states. Manage leads, waitlists, and custom fields. Receive webhook notifications for events across bookings, units, invoices, users, and billing lifecycle changes.
