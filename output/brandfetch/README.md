# <img src="logo.png" height="20"> Brandfetch

Retrieve company brand assets and data by domain, ticker, ISIN, or cryptocurrency symbol. Look up logos (with light/dark themes in SVG, PNG, WebP), color palettes, fonts, images, company descriptions, and firmographic data (employee count, founding year, industry, headquarters). Search brands by name for autocomplete experiences. Resolve raw payment transaction descriptors to merchant brands. Embed brand logos via CDN URLs with customizable dimensions, formats, and fallback options. Monitor brand data changes via webhooks for updates, claims, verifications, and deletions.
