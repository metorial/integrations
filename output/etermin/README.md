# Etermin

Create, read, update, and delete appointments and contacts in an online scheduling platform. Query available time slots and working times for specific calendars and dates. Manage bookable services and calendars. Create vouchers with configurable values, usage limits, and expiration dates. Retrieve company account information. Receive webhook notifications when appointments are created, modified, or deleted. Supports single sign-on token management for enterprise accounts.
