# Printautopilot

Submit PDF documents to remote print queues for automated physical printing. Add PDF files to designated print queues, route documents to specific printers, and retrieve print job statuses. Supports multiple print queues for organizing different document types such as packing slips, shipping labels, and invoices across different printers.
