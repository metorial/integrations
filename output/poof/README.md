# Poof

Process cryptocurrency and fiat payments, create checkout sessions, and manage transactions. Create blockchain wallets, generate deposit addresses, and check balances across 200+ tokens. Send cryptocurrency payouts to external addresses. Create invoices in multiple fiat currencies (USD, EUR, CNY) with cryptocurrency settlement (BTC, LTC, DOGE, ETH). Process ACH bank transfers and check transfer statuses. Fetch ERC-20 token prices, gas prices, and smart contract details. Manage products for checkout flows. Subscribe to webhook notifications for payment events and recurring payment updates.
