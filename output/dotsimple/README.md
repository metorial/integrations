# Dotsimple

Create, schedule, publish, and manage social media posts across multiple platforms including Facebook, Instagram, LinkedIn, X/Twitter, TikTok, Pinterest, YouTube, and Mastodon. Manage connected social accounts, browse and delete media files, create and organize tags with custom colors, and retrieve account-level analytics reports. Supports platform-specific post options (e.g., TikTok privacy, YouTube visibility, Instagram post type), bulk post deletion, publishing queues, and autoresponder management.
