# <img src="logo.svg" height="20"> Pdf.co

Convert, extract data from, edit, and generate PDF documents. Convert PDFs to CSV, JSON, text, Excel, XML, HTML, and images. Generate PDFs from HTML, DOC, images, URLs, and other formats. Merge multiple documents into a single PDF or split PDFs by page ranges. Add text, images, and signatures to PDFs. Fill and extract PDF form fields. Search and replace text within PDFs. Apply OCR to scanned documents. Add or remove PDF passwords and set security permissions. Parse invoices and documents using AI to extract structured data. Classify documents by keyword rules. Generate and read barcodes (QR Code, DataMatrix, Code 128, etc.). Convert Excel files to CSV, HTML, JSON, and other formats. Extract email attachments and send emails.
