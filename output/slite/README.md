# Slite

Create, read, update, and delete notes (documents) in a team knowledge base. Organize documents in parent-child hierarchies, create from templates, and manage document lifecycle including verification, archiving, and ownership assignment. Ask natural language questions to the AI-powered knowledge base. Index custom external content into the AI search index. Search notes by keyword, update individual content tiles within documents, and manage users and groups. List publicly shared, inactive, or empty notes for content auditing.
