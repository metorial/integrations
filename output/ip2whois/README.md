# <img src="logo.png" height="20"> Ip 2 Whois

Look up domain WHOIS records to retrieve registration and ownership information including creation date, expiration date, domain age, registrant contact details, nameservers, and more. Find hosted domains by IP address (IPv4 and IPv6). Convert internationalized domain names to/from punycode, extract domain names from URLs, and identify gTLD or ccTLD extensions. Supports 1221 TLDs and 634 ccTLDs.
