# <img src="logo.png" height="20"> HTML to Image

Convert HTML/CSS markup and public URLs into images (PNG, JPEG, WebP) and PDFs. Generate screenshots of webpages, create dynamic images from reusable templates with Handlebars variable substitution, and manage generated images. Supports Google Fonts, custom viewport dimensions, device scale, render delays, CSS selector cropping, on-the-fly image resizing, and signed URLs for social sharing (og:image, twitter:image). Images are permanently hosted and CDN-cached.
