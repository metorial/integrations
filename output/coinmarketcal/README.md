# <img src="logo.png" height="20"> Coin Market Cal

Search and retrieve cryptocurrency market events from an evidence-based, community-driven economic calendar. Filter events by date range, specific coins, and event categories such as hard forks, exchange listings, airdrops, and partnerships. Sort results by creation date or trending status. List available cryptocurrencies and event categories tracked in the system. Useful for market sentiment analysis, trading signals, and investment research.
