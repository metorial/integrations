# Paperform

Manage online forms, submissions, and eSignature documents. Retrieve and update forms, fields, and submissions. Create and manage products, coupons, and translations for forms. Organize forms into spaces. Access partial (incomplete) submissions. Send documents for eSignature via Papersign, track signing progress, copy and cancel documents, and organize them into folders. Receive webhooks for new submissions, partial submissions, and Papersign document events such as sent, completed, signed, rejected, and expired.
