# <img src="logo.svg" height="20"> Zoho Mail

Send, receive, search, and manage email messages, threads, folders, labels, and signatures. Schedule emails, manage drafts, flag and archive messages, and apply labels. Create and manage tasks with priorities, due dates, reminders, subtasks, and recurring schedules. Create and organize notes into books with attachments. Manage bookmarks in collections. Administer organizations, domains, users, groups, mail policies, and anti-spam settings. Configure email forwarding, vacation replies, domain DNS records (MX, SPF, DKIM), and user protocol access (IMAP, POP, ActiveSync). Retrieve audit logs, login history, and SMTP logs. Receive webhooks for incoming emails, new tasks, and Streams posts.
