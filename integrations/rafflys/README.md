# <img src="logo.png" height="20"> Rafflys

Manage social media giveaways, sweepstakes, and contest promotions across Instagram, Facebook, Twitter/X, YouTube, TikTok, LinkedIn, Bluesky, and Twitch. Pick random winners from comments with custom filters (mentions, hashtags, duplicate exclusion). Collect leads via fortune wheel spin-to-win promotions with registration forms and prize configurations. Retrieve account details, list promotion leads, and poll for newly captured leads. Generate winner certificates and videos for sharing.
