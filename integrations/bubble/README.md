# <img src="logo.jpg" height="20"> Bubble

Perform CRUD operations on Bubble application databases and trigger server-side workflows. Create, read, update, and delete records across exposed data types. Search and filter records by field values, geographic coordinates, and date ranges. Trigger custom API workflows that can create data, sign up or log in users, send emails, and return data. Manage user authentication by signing up and logging in users to obtain session tokens.
