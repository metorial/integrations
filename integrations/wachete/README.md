# Wachete

Monitor web page content for changes and receive notifications when updates occur. Create and manage monitors (wachets) for single pages or entire portals with configurable crawling depth. Track specific page sections using XPath selectors, enable JavaScript rendering for dynamic content, and monitor password-protected pages and documents (PDF, DOC). Configure alert conditions, notification channels (email, webhook), and monitoring frequency. Retrieve change history with diff output, browse notifications filtered by date range, and organize monitors into folders.
