# Castingwords

Order human-powered transcriptions of audio and video files across multiple tiers (Budget, 7 Day, 1 Day) with add-ons like timestamps, captions, and verbatim transcription. Track audiofile status, retrieve completed transcripts in multiple formats (TXT, DOC, RTF, HTML), upgrade or cancel orders, view invoices, and check prepay balance. Receive webhook notifications for transcript completion, duplicate files, refunds, and difficult audio detection.
