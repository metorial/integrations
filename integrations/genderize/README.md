# <img src="logo.svg" height="20"> Genderize

Predict the gender associated with a given name using statistical data. Returns a gender prediction (male or female) with a probability score and data count. Supports country-specific localization using ISO 3166-1 alpha-2 codes for higher accuracy, and batch prediction of up to ten names per request.
