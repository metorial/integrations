# Mailercloud

Create and send email marketing campaigns and transactional emails. Manage contact lists, add and update subscribers, and organize contacts with custom properties. Set up email automations to send targeted, personalized emails based on segments, lists, or webforms. Segment contact lists by criteria like subscription status, demographics, or campaign tags. Track campaign performance through webhooks for events like sends, opens, clicks, bounces, spam reports, and unsubscribes. Create and manage webhooks for real-time event notifications. Retrieve account and plan information.
