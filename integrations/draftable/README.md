# <img src="logo.svg" height="20"> Draftable

Compare two documents to detect differences in text, formatting, and layout. Supports Word, PowerPoint, Excel, PDF, RTF, and text files, including cross-format comparisons (e.g., Word to PDF). Create comparisons by uploading files or providing URLs, retrieve structured change details as JSON, view comparisons in interactive side-by-side or redline viewers, and export comparison results to PDF. Manage comparisons by listing, retrieving, or deleting them.
