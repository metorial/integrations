# <img src="logo.png" height="20"> Emailable

Verify email addresses for deliverability, validity, and safety. Perform single real-time email verification or batch verify up to 50,000 emails at once. Retrieve detailed metadata including deliverability state, risk score, disposable/free/role-based detection, SMTP provider info, suggested spelling corrections, and name/gender inference. Check account credit balance and monitor batch verification status with callback support.
