# Zixflow

Manage CRM data and send multichannel messages via WhatsApp, SMS, RCS, and Email. Create, read, update, and delete collection records, list entries, and custom attributes. Send WhatsApp template and direct messages, SMS (transactional/promotional), RCS rich media messages, and emails with attachments. Deliver OTPs with automatic multi-channel fallback. Manage activities and tasks tied to CRM records. Retrieve campaign delivery reports across channels. Access workspace member information. Receive webhooks for incoming WhatsApp and RCS messages, SMS delivery reports, and CRM data changes.
