# Zulip

Send, retrieve, edit, and delete messages in channels (streams) and direct messages. Manage channels, topics, and user subscriptions. Search and filter messages using narrow queries. Add emoji reactions, upload file attachments, and manage message flags (read, starred). Create and manage channels with public/private visibility, subscribe/unsubscribe users, and configure per-channel notification settings. Retrieve and manage users, user groups, presence status, and typing indicators. Configure organization settings including custom emoji, linkifiers, and custom profile fields. Send invitations, manage drafts and saved snippets, and schedule messages for future delivery. Register for real-time events via long-polling to receive updates on new messages, reactions, presence changes, channel updates, and user activity.
