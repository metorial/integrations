# <img src="logo.svg" height="20"> Shortpixel

Compress, resize, and convert images and PDFs to reduce file sizes. Optimize images via URL or direct upload with lossy, glossy, or lossless compression. Convert images to next-gen formats like WebP and AVIF. Resize and smart-crop images, upscale images by 2x-4x, and remove or replace image backgrounds. Deliver optimized images on-the-fly via CDN with adaptive parameters. Manage account domains, check CDN usage statistics, and monitor available API credits.
