# <img src="logo.jpg" height="20"> Chatwork

Send, read, edit, and delete messages in chat rooms. Create and manage group chat rooms with configurable members and roles. Assign and track tasks within rooms with due dates and completion status. Upload and retrieve files in chat rooms. Manage contacts and handle contact requests. Retrieve user profile information, unread counts, and mention counts. Receive real-time webhooks for new messages, edited messages, and mentions.
