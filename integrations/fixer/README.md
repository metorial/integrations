# <img src="logo.png" height="20"> Fixer

Retrieve real-time and historical foreign exchange rate data for 170+ world currencies. Convert amounts between currencies, query historical rates back to 1999, fetch time-series data for date ranges up to 365 days, and get fluctuation data showing day-to-day currency changes. List all supported currencies including Bitcoin, Gold, and Silver.
