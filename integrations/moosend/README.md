# <img src="logo.png" height="20"> Moosend

Create, send, and manage email marketing campaigns including regular, RSS, A/B testing, and transactional emails. Manage mailing lists, subscribers, custom fields, and segments. Add or remove subscribers, unsubscribe users, and bulk remove contacts. Create audience segments with criteria for targeted campaigns. Send transactional emails with dynamic personalized content. Retrieve campaign analytics including opens, clicks, complaints, and unsubscribes.
