# Shippo

Compare shipping rates across 85+ carriers (USPS, FedEx, UPS, DHL), create and purchase shipping labels, and track packages with normalized status updates. Manage addresses with validation, create shipments, handle customs declarations for international shipping, schedule carrier pickups, process label refunds, and generate manifests. Supports order management, carrier account configuration, parcel templates, batch label operations, and real-time webhook notifications for tracking and transaction events.
