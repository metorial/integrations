# Tripadvisor

Search for hotels, restaurants, and attractions by keyword or geographic coordinates. Retrieve detailed location information including ratings, reviews, photos, addresses, and rankings. Map external property identifiers to Tripadvisor location IDs. Send review collection emails to hotel guests via Review Express. Check hotel opt-in status for review programs. Access TripConnect advertising campaign performance data including click and conversion metrics.
