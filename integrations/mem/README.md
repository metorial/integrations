# Mem

Create, read, search, and delete notes in an AI-powered personal knowledge base. Organize notes into collections, search across notes with filters (by collection, tasks, images, attachments), and use the AI-powered "Mem It" feature to process raw content (web pages, emails, transcripts, articles) into structured knowledge base entries with custom instructions and context. Manage collections by creating, listing, searching, and deleting them.
