# <img src="logo.png" height="20"> Zoho

Manage CRM records including leads, contacts, accounts, and deals. Create, read, update, and delete support tickets in Zoho Desk. Handle accounting operations such as invoices, expenses, payments, purchase orders, and bank transactions in Zoho Books. Manage employee records, leave, attendance, and timesheets in Zoho People. Organize projects, tasks, milestones, timesheets, and bugs in Zoho Projects. Administer email accounts, domains, users, and mail policies in Zoho Mail. Run email marketing campaigns and manage mailing lists in Zoho Campaigns. Import data and create reports and dashboards in Zoho Analytics. Interact with custom applications built in Zoho Creator. Supports webhooks and event notifications across CRM, Desk, Billing, Sign, and ZeptoMail for real-time updates on record changes, ticket events, document actions, and email delivery events. Bulk API available for large data operations in CRM. Query CRM data using COQL (CRM Object Query Language).
