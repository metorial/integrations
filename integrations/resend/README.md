# <img src="logo.png" height="20"> Resend

Send, receive, and manage emails programmatically. Send transactional and marketing emails with HTML, plain text, or React content, including attachments, scheduling, and batch sending. Manage sending domains with DNS verification (DKIM, SPF, DMARC) and open/click tracking. Create and organize contacts into audiences with custom properties, segments, and topic-based subscription preferences. Send broadcast campaigns immediately or on a schedule with automatic unsubscribe handling. Create and manage reusable email templates. Set up automated email workflows triggered by custom events. Manage API keys and webhook endpoints for real-time email, contact, and domain event notifications.
