# <img src="logo.svg" height="20"> Simple Analytics

Retrieve privacy-focused web analytics data including aggregated statistics, event counts, and raw data exports. Query pageviews, visitors, referrers, countries, UTM parameters, browser/OS info, and device types with flexible date ranges and filtering. Track custom event counts with metadata. Export raw page view and event data in CSV or JSON format. Manage tracked websites including adding new sites and assigning labels.
