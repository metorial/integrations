# Twocaptcha

Solve CAPTCHAs and recognize image text using human workers and AI models. Submit captcha tasks (reCAPTCHA V2/V3, Cloudflare Turnstile, FunCaptcha, GeeTest, hCaptcha, and 30+ other types), retrieve solutions via polling or webhook callbacks, and convert arbitrary images to text. Report solution accuracy, check account balance, and configure proxy settings for IP-matched solving. Also provides access to residential proxy services including usage tracking and location browsing.
