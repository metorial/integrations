# <img src="logo.png" height="20"> Never Bounce

Verify individual email addresses in real time and validate bulk email lists. Returns verification results including valid, invalid, disposable, catchall, or unknown statuses along with DNS flags and suggested corrections. Create and manage bulk verification jobs from direct input or remote CSV files, monitor job progress, download results, and receive webhook callbacks on job status changes. Perform free list analysis to estimate bounce rates, check account credit balances, and confirm proof-of-entry verification from the JavaScript widget.
