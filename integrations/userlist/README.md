# <img src="logo.png" height="20"> Userlist

Manage users, companies, and their relationships for SaaS email marketing automation. Create, update, and delete user and company records with custom properties. Track behavioral events performed by users or companies. Send transactional emails and in-app messages using templates or custom content. Manage user subscription preferences for messaging topics. Receive webhook notifications for user, company, relationship, message, and form events.
