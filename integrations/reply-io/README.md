# <img src="logo.svg" height="20"> Reply.io

Manage multichannel sales outreach sequences across email, LinkedIn, calls, SMS, and WhatsApp. Create and configure campaigns with multi-step sequences, add and update contacts, track engagement metrics (opens, clicks, replies, bounces), and manage email templates. Organize contacts into lists, manage blacklists, handle tasks (calls, meetings, to-dos), retrieve email account statuses, and access campaign performance statistics. Receive real-time webhook notifications for email events (sent, replied, opened, clicked, bounced) and contact lifecycle events (added, opted out, finished).
