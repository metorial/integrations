# <img src="logo.svg" height="20"> Geokeo

Convert addresses and place names into geographic coordinates (forward geocoding) and convert latitude/longitude coordinates into human-readable addresses (reverse geocoding). Filter results by country using ISO country codes. Retrieve bounding box coordinates, formatted postal addresses, and distance calculations. Supports JSON and XML response formats with built-in spell check for improved accuracy.
