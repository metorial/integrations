# <img src="logo.svg" height="20"> Typless

Extract structured data from documents such as invoices, receipts, and purchase orders using AI-powered OCR. Upload PDF, JPG, PNG, TIFF, or BMP files and receive extracted field values with confidence scores. Manage document types to configure which fields to extract, build training datasets by adding validated documents, and provide feedback on extraction results to continuously improve model accuracy. Supports 170+ languages and normalizes dates and numbers automatically.
